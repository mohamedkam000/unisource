# Unisource

<img src="assets/header.png" alt="Unisource Banner" width="100%">

Unisource is a lightweight, category-driven Android application designed for fast distribution of academic files such as PDFs, sheets, books, and university announcements. The application allows users to select their college and department on first launch, then provides a clean and modern Material 3 interface for browsing and downloading content.

---

## Features

- Modern Jetpack Compose UI using Material 3
- Categorized content browsing (books, sheets, announcements, etc.)
- Horizontal card lists and vertically scrolling categories
- Smooth image loading with Coil
- Local file access for items stored in the deviceâ€™s Downloads directory
- Highly extensible architecture for future modules

## Getting Started

1. Clone the repository  
   `git clone https://github.com/mohamedkam000/unisource.git`

2. Open the project in Android Studio.

3. Build and Run  
   Use a device or emulator running Android 12.0 (API 31) or above.
